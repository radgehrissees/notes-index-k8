# notes-index-k8

## 项目简介

`notes-index-k8` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不绑定任何特定域名或网站，主要作为个人笔记、文档或静态页面的集中存放与展示空间。

## 目录说明

仓库根目录下包含以下内容：

- `index.html` — 入口页面，用于浏览仓库内所有已归档的 HTML 页面列表
- `pages/` — 存放各独立 HTML 页面的子目录
- `assets/` — 存放页面所需的公共资源（如样式、脚本、图片等）
- `README.md` — 本文件

## 页面归档说明

所有归档的 HTML 页面均以独立的文件形式存放于 `pages/` 目录下。每个页面之间相互独立，不依赖外部服务或数据库。页面内容主要涵盖技术笔记、学习记录、工具演示等，不包含商业推广或诱导点击内容。

如需新增页面，请直接在 `pages/` 下创建新的 `.html` 文件，并在 `index.html` 中添加相应链接即可。
